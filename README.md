# NAME

Cart::Kata - Shopping Cart Exercise

# SYNOPSIS

    use Cart::Kata;

# DESCRIPTION

Cart::Kata offers an implemented solution to the 'cart_kata' Shopping Cart challenge set by SpareRoom, a.k.a. Flatshare Ltd., as part of their recruitment process.
